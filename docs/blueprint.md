# **App Name**: Startup Ally

## Core Features:

- Strategy Planner: AI powered tool that helps founders craft comprehensive business strategies by suggesting relevant frameworks and market insights, using the input of the founder, accessible on /strategy-planner.
- Pitch Deck Generator: Automatically generate a visually appealing pitch deck from strategy data using AI, providing editable templates, accessible on /pitch-deck.
- Financial Summary Tool: Generates key financial reports (revenue projections, cost analysis) and dashboards using uploaded data and AI, available at /financial-summary.
- Investor Materials Compilation: Organizes pitch decks, business plans, and financial summaries into a shareable package for potential investors; accessible on /investor-materials.
- Subscription Management: Handles user subscriptions, billing, and premium feature access, integrates with a payment gateway.
- User Authentication: Provides secure login/signup with role-based access control (founder, admin).
- Dashboard: Personalized dashboard view that organizes quick links to key actions.

## Style Guidelines:

- Primary color: Soft gold (#E4B641) for a touch of luxury and professionalism.
- Background: Off-white (#F8F8F8) for a clean, modern aesthetic.
- Accent color: Deep blue (#1A53BA) for actionable items and interactive elements.
- Body text: 'Inter' sans-serif font for readability.
- Headline text: 'Space Grotesk' sans-serif font for a modern, tech-forward feel.
- Minimalist, line-based icons for navigation and key features.
- Clean, minimalist layout with ample white space.
- Subtle fade-in animations on page load and when new content appears.
